What is a Package

A file archive used to ship applications into the platform.
It contains:

  • A mandatory manifest.
  • Optional icon, libraries, html/css/js files, resources, etc.

Public API overview

  • The User Interfaces which provide the system visual element
  • The Services are background running tasks
  • The Data Sources whose jobs is to provide access the data available on the system
